Aliases = win! + happiness

Aliases allowed you to literally create you own convenient alias to an otherwise long and complicated command as an example:

alias gp='git push origin master'

But even better aliases can be concatenated by using the ; operator in order to create a chain of commands that need to be run in an specific order.

alias cdt='cd; cd Documents/Tealeaf/'

The previous code will work regardeless the directory I am working on, becuase it will first cd to the root directory with cd and then it will get me to the Tealeaf directory with cd Document/Tealeaf/

Simple functions

You can also create functions that take parameters (such as folder names) and make it perform different actions.

function mcd() {
	mkdir -p "$1" && cd "$1";
}

This function creates a directory with a certain name and immediately cd into it, which something I do fairly often.
