The Fabrication and Faker gem combo allows us to automate the creation of multiple instances of our models. This is a great time saver when one have to create tests or seed the database

In order to set the combo we need to:

1. Install Gems

#gemfile
gem 'Fabrication'
gem 'Faker'

2. Create the corresponding directories and files

# spec/fabricators/todo_fabricators.rb
Fabricator(:todo) do
  name {'cooking'}
  # if we want to associate an user object to the todo 
  user
end

Now if we want to set random attributes we can use the fakers modules

Fabricator(:todo) do
  name { Faker::Lorem.words(5).join(' ')} 
  description (Faker::Lorem.paragraphs(2))
  user
end

3. Set the variables

Now we can just simple create random instances of Todo by 

todo = Fabricate(:todo)
